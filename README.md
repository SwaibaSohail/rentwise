# RentWise

Multi-role rental-management app (landlords + tenants). Portfolio project.

**Stack:** React + Vite + Tailwind/shadcn · Node + Express · Socket.io · Postgres (Neon) + Prisma · Firebase Auth.

## Dev

```bash
npm install            # installs all workspaces
npm run dev:server     # API on :4000
npm run dev:client     # web on :5173
```

See `docs/superpowers/specs/` for the design and `docs/superpowers/plans/` for build plans.
